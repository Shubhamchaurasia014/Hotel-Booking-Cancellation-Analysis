# 🏨 Hotel Booking Cancellation Analysis

## 📌 Overview

This project analyzes hotel booking data to understand **customer behavior and booking cancellations**. The goal is to identify key factors that lead to cancellations and derive actionable insights for better decision-making.

---

## 🎯 Objectives

* Analyze overall **cancellation trends**
* Compare cancellations across **hotel types**
* Study impact of factors like:

  * Lead time
  * Deposit type
  * Booking patterns
* Generate insights to reduce cancellation rates

---

## 📂 Dataset

* Source: Hotel Booking Dataset (CSV file)
* Contains booking details such as:

  * Hotel type
  * Lead time
  * Customer details
  * Booking status (Canceled / Not Canceled)

---

## 🛠️ Tech Stack

* **Python**
* **Pandas & NumPy** → Data cleaning & analysis
* **Matplotlib & Seaborn** → Data visualization
* **Jupyter Notebook**

---

## 🔍 Project Workflow

### 1. Data Loading

* Imported dataset using Pandas
* Checked dataset shape and preview

### 2. Data Cleaning

* Handled missing values
* Checked data types and unique values

### 3. Exploratory Data Analysis (EDA)

* Overall cancellation rate
* Cancellation by hotel type
* Lead time distribution
* Visual insights using plots

### 4. Key Performance Indicators (KPIs)

* Cancellation % by hotel type
* Average lead time (canceled vs non-canceled)
* Impact of deposit type on cancellations
* Monthly cancellation trends

---

## 📊 Key Insights

* Higher **lead time** often leads to more cancellations
* Certain **deposit types** significantly affect cancellation behavior
* Cancellation rates vary across **hotel categories**
